WORKFLOW 1: USER SETUP & RECIPE DISCOVERY
------------------------------------------
PANTRYPAL – WEB APPLICATION WORKFLOWS
WORKFLOW 1: USER & PANTRY MANAGEMENT + RECIPE SEARCH

User Journey: “I want to find recipes I can make right now”

FEATURES

User System

Register / Login using email & password

Create and manage user profile

Set dietary restrictions (vegan, vegetarian, gluten-free, allergies)

Select preferred cuisines (Italian, Mexican, Asian, etc.)

Define cooking skill level (Beginner, Intermediate, Advanced)

Pantry Management

Add ingredients manually

Search and select ingredients from a predefined database

Assign quantities to each ingredient

Categorize ingredients (Fridge, Pantry, Freezer)

Edit ingredient details (quantity, category)

Remove ingredients from pantry

Recipe Search & Filtering

Search recipes based on available pantry ingredients

Filter recipes by dietary restrictions

Filter by cuisine type

Filter by difficulty level

Sort results by “most ingredients matching”

View recipes that require minimal missing ingredients

Recipe Viewing

View complete recipe details

Display ingredient list with required quantities

Highlight missing vs available ingredients

Step-by-step cooking instructions

“Start Cooking” button to enter Interactive Cooking Mode
